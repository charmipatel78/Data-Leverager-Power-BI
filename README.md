# 📊 Data Leverager - Power BI Project

## 📌 Overview

This project focuses on data analysis using Power BI. It involves data transformation, modeling, and basic analysis to extract meaningful insights from multiple Excel datasets.

---

## 🎯 Objective

* To clean and transform raw data
* To integrate multiple datasets
* To perform data modeling
* To extract useful insights

---

## 📂 Project Files

* `Data Leverager.pbix` – Power BI project file
* `Employee_Data.xlsx` – Employee dataset
* `Pivot_Unpivot_Data.xlsx` – Data transformation dataset
* `Sales_Jan.xlsx`, `Sales_Feb.xlsx`, `Sales_Mar.xlsx`, `Sales_Apr.xlsx` – Monthly sales data

---

## ⚙️ Key Features

* Data cleaning using Power Query
* Data transformation (Pivot / Unpivot)
* Combining multiple datasets
* Data modeling and relationships
* Basic data analysis

---

## 📈 Key Insights

* Combined monthly sales data for analysis
* Structured raw data into usable format
* Identified important data patterns
* Improved data understanding

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query
* Microsoft Excel

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Refresh data if needed
4. Explore the data model and transformations

---

## 📌 Conclusion

This project demonstrates foundational Power BI skills including data preparation, transformation, and modeling, which are essential for building advanced dashboards and reports.
